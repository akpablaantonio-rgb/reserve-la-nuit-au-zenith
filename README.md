# La Nuit au Zénith

Un site web moderne, élégant et immersif pour l'événement musical et gastronomique "La Nuit au Zénith".

## Description

Ce site web présente une expérience luxueuse et raffinée pour promouvoir l'événement "La Nuit au Zénith". Il comprend trois pages principales : Accueil/Réservation, Confirmation, et Menu de la soirée.

## Fonctionnalités

- **Design haut de gamme** : Palette de couleurs noire, dorée et blanche avec effets glassmorphism
- **Animations fluides** : Particules flottantes, bulles, effets de lumière dorée
- **Formulaire de réservation** : Validation JavaScript et stockage local
- **Navigation responsive** : Optimisé pour mobile, tablette et desktop
- **Effets visuels** : Parallax, hover effects, transitions élégantes
- **Bouton WhatsApp flottant** : Contact direct avec l'organisateur

## Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Design, animations, responsive design
- **JavaScript** : Interactions, validations, animations dynamiques

## Structure du projet

```
la-nuit-au-zenith/
├── index.html          # Page d'accueil et réservation
├── confirmation.html   # Page de confirmation
├── menu.html           # Page du menu
├── style.css           # Styles CSS
└── script.js           # JavaScript
```

## Comment utiliser

1. Ouvrez `index.html` dans votre navigateur web
2. Remplissez le formulaire de réservation
3. Cliquez sur "Réserver maintenant"
4. Vous serez redirigé vers la page de confirmation
5. Depuis la confirmation, vous pouvez accéder au menu

## Fonctionnalités JavaScript

- Validation du formulaire
- Stockage des données de réservation en localStorage
- Animations au scroll
- Effets de particules et bulles
- Navigation entre pages
- Ouverture de WhatsApp

## Responsive Design

Le site est entièrement responsive et s'adapte aux :
- Écrans de bureau (>768px)
- Tablettes (768px - 480px)
- Téléphones (<480px)
